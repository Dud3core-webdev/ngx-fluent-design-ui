ngx-fluent-design

An open-source, Angular implementation of Microsoft's Fluent Design Library

This library has only so far been tested with Angular 11. It may work with newer and some older versions of Angular however.

Getting started

To get started with this library, run either:


yarn add ngx-fluent-design

or

npm i ngx-fluent-design --save

to install the package locally.

Setting the project up for usage
